\IDS_Monitor

Monitoring engine

This class represents the core of the frameworks attack detection mechanism
and provides functions to scan incoming data for malicious appearing script
fragments.

Methods

private

  • _detect() — Checks whether given value matches any of the supplied filter patterns
  • _diff() — This method calculates the difference between the original and the purified markup strings.
  • _iterate() — Iterates through given data and delegates it to IDS_Monitor::_detect() in order to check for malicious appearing fragments
  • _jsonConcatContents() — This is the callback used in _jsonDecodeValues(). The method concatenates key and value and stores them in $this->tmpJsonString.
  • _jsonDecodeValues() — This method prepares incoming JSON data for the PHPIDS detection process. It utilizes _jsonConcatContents() as callback and returns a string version of the JSON data structures.
  • _match() — Matches given value and/or key against given filter
  • _purifyValues() — Purifies given key and value variables using HTMLPurifier
